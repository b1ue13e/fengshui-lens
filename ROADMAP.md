# ROADMAP

> 目标：把 **青年大学习** 打磨成一个真正有产品感、可传播、值得被 star 的中文开源产品仓库。  
> 不是做一个大而全的内容平台，而是做一个 **高频现实决策训练系统**。

---

# North Star

## 1. 产品 North Star

让第一次独立面对社会现实的年轻人，能够：

- 更快进入判断
- 更少依赖猜测
- 更少踩真坑
- 更容易继续行动

## 2. 开源 North Star

让仓库具备：

- 一眼能理解的产品定位
- 有辨识度的前台体验
- 有完整主线的使用路径
- 有演示价值和传播价值
- 最终冲到 **10k stars** 的潜力

# 10k Stars 路线图

如果目标真的是 **10k stars**，我认为不是靠“多做功能”，而是靠以下几件事：

## 第一阶段：先做到 100 stars
核心靠：

- README 足够强
- 产品主线能跑
- 截图清楚
- 一眼看懂价值

## 第二阶段：冲到 1k stars
核心靠：

- 产品完成度上升
- 页面统一度更高
- 更适合被转发和讨论
- 有明确的产品观点和审美方向

## 第三阶段：冲到 10k stars
核心靠：

- 项目不只是“能跑”，而是“值得记住”
- 形成独特定位：**中文世界里少见的产品型现实决策训练仓库**
- 有持续内容、持续迭代、持续传播

# 当前不做什么

为了保证项目收口，以下方向暂时不做：

- [ ] 做成泛内容门户
- [ ] 做成信息流产品
- [ ] 首页堆几十个入口
- [ ] 上来就支持全国城市
- [ ] 过早做复杂权限 / 组织 / 平台系统
- [ ] 为了“显得大”而堆很多空功能
